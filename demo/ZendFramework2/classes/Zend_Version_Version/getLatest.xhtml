Zend\Version\Version::getLatest

Fetches the version of the latest stable release.

By default, this uses the API provided by framework.zend.com for version retrieval. If $service is set to VERSION_SERVICE_GITHUB, this will use the GitHub API (v3) and only returns refs that begin with * 'tags/release-'. Because GitHub returns the refs in alphabetical order, we need to reduce the array to a single value, comparing the version numbers with version_compare().
