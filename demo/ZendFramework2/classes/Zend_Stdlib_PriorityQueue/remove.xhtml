Zend\Stdlib\PriorityQueue::remove

Remove an item from the queue

This is different than ; its purpose is to dequeue an item. This operation is potentially expensive, as it requires re-initialization and re-population of the inner queue. Note: this removes the first item matching the provided item found. If the same item has been added multiple times, it will not remove other instances.
