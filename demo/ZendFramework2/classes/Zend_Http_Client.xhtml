    • publicsetFileUpload( string $filename, string $formname, string $data, string $ctype )

    • Set a file to upload (using a POST request)
      Can be used in two ways:

1. $data is null (default): $filename is treated as the name if a local file which
will be read and sent. Will try to guess the content type using mime_content_type().
2. $data is set - $filename is sent as the file name, but $data is sent as the file
contents and no file is read from the file system. In this case, you need to
manually set the Content-Type ($ctype) or it will default to
application/octet-stream.

        • Parameters:

        • $filename - Name of file to upload, or name to save as
        • $formname - Name of form element to send as
        • $data - Data to send (if null, $filename is read and sent)
        • $ctype - Content type to use (if $data is set and $ctype is

        Returns:

        Throws:

    • protecteddetectFileMimeType( string $file )

    • Attempt to detect the MIME type of a file using available extensions
      This method will try to detect the MIME type of a file. If the fileinfo
extension is available, it will be used. If not, the mime_magic
extension which is deprecated but is still available in many PHP setups
will be tried.

If neither extension is available, the default application/octet-stream
MIME type will be returned

        • Parameters:

        • $file - File path

        Returns:

        • MIME type
