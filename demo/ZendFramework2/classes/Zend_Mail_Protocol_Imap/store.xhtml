Zend\Mail\Protocol\Imap::store

set flags

    Signature

    public function store(array $flags, [int $from = null, [int|null $to = null, [string|null $mode = null, [bool $silent = true]]]] )

    Parameters

    $flags — array
    flags to set, add or remove - see $mode
    $from — int
    message for items or start message if $to !== null
    $to — object
    if null only one message ($from) is fetched, else it's the
    last message, INF means last message available
    $mode — object
    '+' to add flags, '-' to remove flags, everything else sets the flags as given


    $silent — object
    if false the return values are the new flags for the wanted messages


    Returns

    bool|array
    new flags if $silent is false, else true or false depending on success
