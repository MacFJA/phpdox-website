Zend\Navigation\Page\AbstractPage::__isset

Checks if a property is set

Magic overload for enabling <code>isset($page->propname)</code>. Returns true if the property is native (id, class, title, etc), and true or false if it's a custom property (depending on whether the property actually is set).
