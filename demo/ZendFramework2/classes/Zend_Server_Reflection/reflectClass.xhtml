Zend\Server\Reflection::reflectClass

Perform class reflection to create dispatch signatures

Creates a object for the class or object provided. If extra arguments should be passed to dispatchable methods, these may be provided as an array to $argv.

    Signature

    public function reflectClass(string|object $class, [bool|array $argv = false, [string $namespace = '']] )

    Parameters

    $class — object
    Class name or object


    $argv — object
    Optional arguments to be used during the method call


    $namespace — string
    Optional namespace with which to prefix the
    method name (used for the signature key). Primarily to avoid collisions,
    also for XmlRpc namespacing

    Returns

    ReflectionClass
