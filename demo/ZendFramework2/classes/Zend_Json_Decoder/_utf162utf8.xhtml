Zend\Json\Decoder::_utf162utf8

Convert a string from one UTF-16 char to one UTF-8 char.

Normally should be handled by mb_convert_encoding, but provides a slower PHP-only method for installations that lack the multibyte string extension. This method is from the Solar Framework by Paul M. Jones
