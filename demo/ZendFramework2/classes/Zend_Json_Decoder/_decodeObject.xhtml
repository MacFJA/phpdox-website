Zend\Json\Decoder::_decodeObject

Decodes an object of the form: { "attribute: value, "attribute2" : value,...}

If Zend\Json\Encoder was used to encode the original object then a special attribute called __className which specifies a class name that should wrap the data contained within the encoded source. Decodes to either an array or stdClass object, based on the value of {@link $decodeType}. If invalid $decodeType present, returns as an array.
