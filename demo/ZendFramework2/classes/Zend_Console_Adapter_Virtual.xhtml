    • Write a box at the specified coordinates.
      If X or Y coordinate value is negative, it will be calculated as the distance from far right or bottom edge

        • Parameters:

        • $x1 - Top-left corner X coordinate (column)

        • $y2 - Bottom-right corner Y coordinate (row)
        • $lineStyle - (optional) Box border style.
        • $fillStyle - (optional) Box fill style or a single character to fill it with.
        • $color - (optional) Foreground color
        • $bgColor - (optional) Background color
        • $fillColor - (optional) Foreground color of box fill
        • $fillBgColor - (optional) Background color of box fill

        Throws:

        • Exception\BadMethodCallException if coordinates are invalid

    • publicwriteTextBlock( string $text, int $width, int|null $height, integer $x, integer $y, null|int $color, null|int $bgColor )

    • Write a block of text at the given coordinates, matching the supplied width and height.
      In case a line of text does not fit desired width, it will be wrapped to the next line.
In case the whole text does not fit in desired height, it will be truncated.

        • Parameters:

        • $text - Text to write
        • $width - Maximum block width. Negative value means distance from right edge.
        • $height - Maximum block height. Negative value means distance from bottom edge.
        • $x - Block X coordinate (column)
        • $y - Block Y coordinate (row)
        • $color - (optional) Text color
        • $bgColor - (optional) Text background color
