Zend\Mail\Storage\Part\PartInterface::getHeaderField

Get a specific field from a header like content type or all fields as array

If the header occurs more than once, only the value from the first header is returned. Throws an exception if the requested header does not exist. If the specific header field does not exist, returns null.

    Signature

    public function getHeaderField(string $name, [string $wantedPart = '0', [string $firstName = '0']] )

    Parameters

    $name — string
    name of header, like in getHeader()
    $wantedPart — string
    the wanted part, default is first, if null an array with all parts is returned
    $firstName — string
    key name for the first part

    Returns

    string|array
    wanted part or all parts as array($firstName => firstPart, partname => value)

    Errors/Exceptions

    ExceptionInterface
