Symfony\Component\Validator\ExecutionContextInterface::getMetadata

Returns the metadata for the currently validated value.

With the core implementation, this method returns a instance if the current value is an object, a instance if the current value is the value of a property and a instance if the validated value is the result of a getter method. If the validated value is neither of these, for example if the validator has been called with a plain value and constraint, this method returns null.
