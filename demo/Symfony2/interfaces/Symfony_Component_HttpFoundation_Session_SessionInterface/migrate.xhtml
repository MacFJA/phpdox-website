Symfony\Component\HttpFoundation\Session\SessionInterface::migrate

Migrates the current session to a new session id while maintaining all session attributes.

    Signature

    public function migrate([Boolean $destroy = false, [integer $lifetime = null]] )

    Parameters

    $destroy — object
    Whether to delete the old session or leave it to garbage collection.


    $lifetime — integer
    Sets the cookie lifetime for the session cookie. A null value
    will leave the system settings unchanged, 0 sets the cookie
    to expire with browser session. Time is in seconds, and is
    not a Unix timestamp.

    Returns

    Boolean
    True if session migrated, false if error.
