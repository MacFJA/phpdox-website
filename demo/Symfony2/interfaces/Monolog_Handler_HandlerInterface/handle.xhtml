Monolog\Handler\HandlerInterface::handle

Handles a record.

All records may be passed to this method, and the handler should discard those that it does not want to handle. The return value of this function controls the bubbling process of the handler stack. Unless the bubbling is interrupted (by returning true), the Logger class will keep on calling further handlers in the stack with a given log record.

    Signature

    public function handle(array $record )

    Parameters

    $record — array
    The record to handle

    Returns

    Boolean
    True means that this handler handled the record, and that bubbling is not permitted.
    False means the record was either not processed or that this handler allows bubbling.
