Symfony\Component\Form\FormRendererInterface::searchAndRenderBlock

Searches and renders a block for a given name suffix.

The block is searched by combining the block names stored in the form view with the given suffix. If a block name is found, that block is rendered. If this method is called recursively, the block search is continued where a block was found before.
