Symfony\Component\Intl\DateFormatter\IntlDateFormatter::parse

Parse string to a timestamp value

    Signature

    public function parse(string $value, [int& $position = null] )

    Parameters

    $value — string
    String to convert to a time value
    $position — int
    Not supported. Position at which to start the parsing in $value (zero-based).
    If no error occurs before $value is consumed, $parse_pos will
    contain -1 otherwise it will contain the position at which parsing
    ended. If $parse_pos > strlen($value), the parse fails immediately.

    Returns

    string
    Parsed value as a timestamp

    Errors/Exceptions

    MethodArgumentNotImplementedException
    When $position different than null, behavior not implemented
