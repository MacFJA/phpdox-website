Doctrine\ORM\QueryBuilder::where

Specifies one or more restrictions to the query result.

Replaces any previously specified restrictions, if any. <code> $qb = $em->createQueryBuilder() ->select('u') ->from('User', 'u') ->where('u.id = ?'); // You can optionally programatically build and/or expressions $qb = $em->createQueryBuilder(); $or = $qb->expr()->orx(); $or->add($qb->expr()->eq('u.id', 1)); $or->add($qb->expr()->eq('u.id', 2)); $qb->update('User', 'u') ->set('u.password', md5('password')) ->where($or); </code>
