Doctrine\ORM\QueryBuilder::setParameter

Sets a query parameter for the query being constructed.

<code> $qb = $em->createQueryBuilder() ->select('u') ->from('User', 'u') ->where('u.id = :user_id') ->setParameter('user_id', 1); </code>
