Symfony\Component\Form\Form::offsetSet

Adds a child to the form (implements the \ArrayAccess interface).

    Signature

    public function offsetSet(string $name, FormInterface $child )

    Parameters

    $name — string
    Ignored. The name of the child is used.
    $child — object
    The child to be added.


    Errors/Exceptions

    AlreadySubmittedException
    If the form has already been submitted.
    LogicException
    When trying to add a child to a non-compound form.
