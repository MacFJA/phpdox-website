across several different php projects and several different authors. The

  • camelize() — Camelize a word. This uses the classify() method and turns the first character to lowercase
  • classify() — Convert a word in to the format for a Doctrine class name. Converts 'table_name' to 'TableName'
  • tableize() — Convert word in to the format for a Doctrine table name. Converts 'ModelName' to 'model_name'
