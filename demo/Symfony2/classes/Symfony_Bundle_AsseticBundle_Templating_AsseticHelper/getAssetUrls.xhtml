Symfony\Bundle\AsseticBundle\Templating\AsseticHelper::getAssetUrls

Gets the URLs for the configured asset.

Usage looks something like this: <?php foreach ($view['assetic']->assets('@jquery, js/src/core/*', '?yui_js') as $url): ?> <script src="<?php echo $url ?>" type="text/javascript"></script> <?php endforeach; ?> When in debug mode, the helper returns an array of one or more URLs. When not in debug mode it returns an array of one URL.
