Symfony\Component\DependencyInjection\Extension\Extension::getAlias

Returns the recommended alias to use in XML.

This alias is also the mandatory prefix to use when using YAML. This convention is to remove the "Extension" postfix from the class name and then lowercase and underscore the result. So: AcmeHelloExtension becomes acme_hello This can be overridden in a sub-class to specify the alias manually.
