Symfony\Component\Security\Acl\Dbal\AclProvider::hydrateObjectIdentities

This method is called to hydrate ACLs and ACEs.

This method was designed for performance; thus, a lot of code has been inlined at the cost of readability, and maintainability. Keep in mind that changes to this method might severely reduce the performance of the entire ACL system.
