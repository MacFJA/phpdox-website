Symfony\Component\Form\Extension\Core\ChoiceList\SimpleChoiceList::__construct

Creates a new simple choice list.

    Signature

    public function __construct(array $choices, array $preferredChoices )

    Parameters

    $choices — array
    The array of choices with the choices as keys and
    the labels as values. Choices may also be given
    as hierarchy of unlimited depth by creating nested
    arrays. The title of the sub-hierarchy is stored
    in the array key pointing to the nested array.
    $preferredChoices — array
    A flat array of choices that should be
    presented to the user with priority.
