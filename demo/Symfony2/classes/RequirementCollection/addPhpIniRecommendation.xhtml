RequirementCollection::addPhpIniRecommendation

Adds an optional recommendation in form of a php.ini configuration.

    Signature

    public function addPhpIniRecommendation(string $cfgName, [Boolean|callback $evaluation = false, [Boolean $approveCfgAbsence = false, [string $testMessage = null, [string $helpHtml = null, [string|null $helpText = null]]]]] )

    Parameters

    $cfgName — string
    The configuration name used for ini_get()
    $evaluation — object
    Either a Boolean indicating whether the configuration should evaluate to true or false,
    a callback function receiving the configuration value as parameter to determine the fulfillment of the requirement
    $approveCfgAbsence — object
    If true the Requirement will be fulfilled even if the configuration option does not exist, i.e. ini_get() returns false.
    is is helpful for abandoned configs in later PHP versions or configs of an optional extension, like Suhosin.
    ample: You require a config to be true but PHP later removes this config and defaults it to true internally.
    $testMessage — string
    The message for testing the requirement (when null and $evaluation is a Boolean a default message is derived)
    $helpHtml — string
    The help text formatted in HTML for resolving the problem (when null and $evaluation is a Boolean a default help is derived)
    $helpText — object
    The help text (when null, it will be inferred from $helpHtml, i.e. stripped from HTML tags)
