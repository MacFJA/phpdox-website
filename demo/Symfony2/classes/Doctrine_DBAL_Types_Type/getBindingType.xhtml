Doctrine\DBAL\Types\Type::getBindingType

Gets the (preferred) binding type for values of this type that can be used when binding parameters to prepared statements.

This method should return one of the PDO::PARAM_* constants, that is, one of: PDO::PARAM_BOOL PDO::PARAM_NULL PDO::PARAM_INT PDO::PARAM_STR PDO::PARAM_LOB
