Symfony\Bundle\FrameworkBundle\Templating\Helper\FormHelper::form

Renders the HTML for a form.

Example usage: <?php echo view['form']->form($form) ?> You can pass options during the call: <?php echo view['form']->form($form, array('attr' => array('class' => 'foo'))) ?> <?php echo view['form']->form($form, array('separator' => '+++++')) ?> This method is mainly intended for prototyping purposes. If you want to control the layout of a form in a more fine-grained manner, you are advised to use the other helper methods for rendering the parts of the form individually. You can also create a custom form theme to adapt the look of the form.
