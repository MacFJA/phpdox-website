Assetic\Factory\AssetFactory::createAsset

Creates a new asset.

Prefixing a filter name with a question mark will cause it to be omitted when the factory is in debug mode. Available options: * output: An output string * name: An asset name for interpolation in output patterns * debug: Forces debug mode on or off for this asset * root: An array or string of more root directories
