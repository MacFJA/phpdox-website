Assetic\Factory\AssetFactory::parseInput

Parses an input string string into an asset.

The input string can be one of the following: * A reference: If the string starts with an "at" sign it will be interpreted as a reference to an asset in the asset manager * An absolute URL: If the string contains "://" or starts with "//" it will be interpreted as an HTTP asset * A glob: If the string contains a "*" it will be interpreted as a glob * A path: Otherwise the string is interpreted as a filesystem path Both globs and paths will be absolutized using the current root directory.
