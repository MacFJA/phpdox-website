Symfony\Bundle\FrameworkBundle\Controller\RedirectController::urlRedirectAction

Redirects to a URL.

The response status code is 302 if the permanent parameter is false (default), and 301 if the redirection is permanent. In case the path is empty, the status code will be 404 when permanent is false and 410 otherwise.
