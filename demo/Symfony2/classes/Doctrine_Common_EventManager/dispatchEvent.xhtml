Doctrine\Common\EventManager::dispatchEvent

Dispatches an event to all registered listeners.

    Signature

    public function dispatchEvent(string $eventName, [EventArgs $eventArgs = null] )

    Parameters

    $eventName — string
    The name of the event to dispatch. The name of the event is
    the name of the method that is invoked on listeners.
    $eventArgs — Doctrine\Common\EventArgs
    The event arguments to pass to the event handlers/listeners.
    If not supplied, the single empty EventArgs instance is used.

    Returns

    boolean
