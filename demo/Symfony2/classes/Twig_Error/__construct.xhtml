Twig_Error::__construct

Constructor.

Set both the line number and the filename to false to disable automatic guessing of the original template name and line number. Set the line number to -1 to enable its automatic guessing. Set the filename to null to enable its automatic guessing. By default, automatic guessing is enabled.
