Symfony\Component\Form\Extension\Core\ChoiceList\ObjectChoiceList::__construct

Creates a new object choice list.

    Signature

    public function __construct(Traversable $choices, [PropertyAccessorInterface $labelPath = null, [PropertyAccessorInterface $preferredChoices = null, [PropertyAccessorInterface $groupPath = null, [PropertyAccessorInterface $valuePath = null, [PropertyAccessorInterface $propertyAccessor = null]]]]] )

    Parameters

    $choices — object
    The array of choices. Choices may also be given
    as hierarchy of unlimited depth by creating nested
    arrays. The title of the sub-hierarchy can be
    stored in the array key pointing to the nested
    array. The topmost level of the hierarchy may also
    be a \Traversable.
    $labelPath — string
    A property path pointing to the property used
    for the choice labels. The value is obtained
    by calling the getter on the object. If the
    path is NULL, the object's __toString() method
    is used instead.
    $preferredChoices — array
    A flat array of choices that should be
    presented to the user with priority.
    $groupPath — string
    A property path pointing to the property used
    to group the choices. Only allowed if
    the choices are given as flat array.
    $valuePath — string
    A property path pointing to the property used
    for the choice values. If not given, integers
    are generated instead.
    $propertyAccessor — Symfony\Component\PropertyAccess\PropertyAccessorInterface
    The reflection graph for reading property paths.
