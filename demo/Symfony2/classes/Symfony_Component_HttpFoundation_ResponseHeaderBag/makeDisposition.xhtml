Symfony\Component\HttpFoundation\ResponseHeaderBag::makeDisposition

Generates a HTTP Content-Disposition field-value.

    Signature

    public function makeDisposition(string $disposition, [string $filename = '', [string $filenameFallback = '']] )

    Parameters

    $disposition — string
    One of "inline" or "attachment"
    $filename — string
    A unicode string
    $filenameFallback — string
    A string containing only ASCII characters that
    is semantically equivalent to $filename. If the filename is already ASCII,
    it can be omitted, or just copied from $filename

    Returns

    string
    A string suitable for use as a Content-Disposition field-value.

    Errors/Exceptions

    InvalidArgumentException
