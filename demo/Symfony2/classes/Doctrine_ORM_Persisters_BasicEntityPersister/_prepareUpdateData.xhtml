Doctrine\ORM\Persisters\BasicEntityPersister::_prepareUpdateData

Prepares the changeset of an entity for database insertion (UPDATE).

The changeset is obtained from the currently running UnitOfWork. During this preparation the array that is passed as the second parameter is filled with <columnName> => <value> pairs, grouped by table name. Example: <code> array( 'foo_table' => array('column1' => 'value1', 'column2' => 'value2', ...), 'bar_table' => array('columnX' => 'valueX', 'columnY' => 'valueY', ...), ... ) </code>
