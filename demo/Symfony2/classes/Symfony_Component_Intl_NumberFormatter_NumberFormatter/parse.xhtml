Symfony\Component\Intl\NumberFormatter\NumberFormatter::parse

Parse a number

    Signature

    public function parse(string $value, [int& $type = self::TYPE_DOUBLE, [int& $position = null]] )

    Parameters

    $value — string
    The value to parse
    $type — int
    Type of the formatting, one of the format type constants.
    The only currently supported types are NumberFormatter::TYPE_DOUBLE,
    NumberFormatter::TYPE_INT32 and NumberFormatter::TYPE_INT64.
    $position — int
    Not supported. Offset to begin the parsing on return this value will hold the offset at which the parsing ended

    Returns

    Boolean|string
    The parsed value of false on error

    Errors/Exceptions

    MethodArgumentNotImplementedException
    When $position different than null, behavior not implemented
