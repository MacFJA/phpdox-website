Symfony\Component\Intl\NumberFormatter\NumberFormatter::create

Static constructor.

    Signature

    public function create([string $locale = 'en', [int $style = null, [string $pattern = null]]] )

    Parameters

    $locale — string
    The locale code. The only supported locale is "en".
    $style — int
    Style of the formatting, one of the format style constants.
    The only currently supported styles are NumberFormatter::DECIMAL
    and NumberFormatter::CURRENCY.
    $pattern — string
    Not supported. A pattern string in case $style is NumberFormat::PATTERN_DECIMAL or
    NumberFormat::PATTERN_RULEBASED. It must conform to the syntax
    described in the ICU DecimalFormat or ICU RuleBasedNumberFormat documentation

    Returns

    NumberFormatter
