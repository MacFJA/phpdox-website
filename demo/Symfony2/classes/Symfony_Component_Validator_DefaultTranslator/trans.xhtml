Symfony\Component\Validator\DefaultTranslator::trans

Interpolates the given message.

Parameters are replaced in the message in the same manner that {@link strtr()} uses. Example usage: $translator = new DefaultTranslator(); echo $translator->trans( 'This is a {{ var }}.', array('{{ var }}' => 'donkey') ); // -> This is a donkey.
